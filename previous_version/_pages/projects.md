## [OptKit](https://github.com/ferdous-alam/OptKit) 
Open source python library for mathematical optimization. The codes are supposed to be modular and self-containing. This is an ongoing project. 

## [Deconstructed ML](https://github.com/ferdous-alam/Deconstructed_ML) 
This is an ongoing effort to deconstruct state-of-the-art deep learning models and analyze exactly why and how they work. Useful data visualizations are supposed to be used to make these models more intuitive. I also plan to maintain a colab explaining the anatomy of these models. 


## [Deconstructed RL](https://github.com/ferdous-alam/Deconstructed_RL)
This is also an ongoing project on recent RL, some people like to call it "Deep RL", algorithms. I am planning to make modular implementation of classic algorithms including but not limited to the following; value based: Q-learning, DQN, DDPG, policy based: REINFORCE, vanilla policy gradient, TRPO, PPO, A3C etc. 


## [Applied ML tutorials](https://github.com/ferdous-alam/Applied-Machine-Learning-Tutorials)
Tutorials on statistical learning algorithms. Classic machine learning algorithms like linear and polynomial regression, logistic regression, LDA, QDA, PCA, SVM, Tree etc. are demonstrated in jupyter notebooks. The codes are self-containing. The motivation of this tutorial comes from the classic "Probabilistic Machine Learning" book by Kevin Murphy.
